Follow these steps to create an NPC using the NPC Quick Mode of RPG Lore Weaver.

1. **Load the skill**
   - Read `rpg-lore-weaver/SKILL.md` and `rpg-lore-weaver/references/npc-quick-mode.md`.

2. **Rapid-Fire Input**
   - Ask the user for: Name (or generate), Role, System & Setting, Tone.
   - This should be quick — 4 questions max.

3. **3-Pillar Generation**
   - Generate the NPC using the 3-pillar system from `npc-quick-mode.md`:
     - Pillar 1: Hook (one-line concept)
     - Pillar 2: Want & Crack (motivation + vulnerability)
     - Pillar 3: DM Notes (secret, escalation, connection to PCs)

4. **Tiered Output**
   - Present Tier 1 (minimal). Ask: "Want more depth?"
   - If yes, expand to Tier 2 (expanded) or Tier 3 (full NPC).

5. **Validate**
   - Run the output through the NPC validation checklist mentally.
   - Present the final NPC to the user.
