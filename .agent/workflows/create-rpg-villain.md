---

Follow these steps to create a villain using the Villain Mode of RPG Lore Weaver.

1. **Load the skill**
   - Read `rpg-lore-weaver/SKILL.md` and `rpg-lore-weaver/references/villain-mode.md`.

2. **V1: The Seed**
   - Ask the user: What role does this villain play? (BBEG, recurring rival, corrupted ally, faction leader, force of nature)
   - Ask: System & setting, tone, how personal is the conflict with the PCs?

3. **V2: The Mirror (Motivation)**
   - Build the villain's motivation: The Wound, The Logic, The Line, Push vs Pull.
   - Remember: "They're just evil" is NOT a motivation.

4. **V3: The Plan**
   - Define: Goal, Method, Timeline, Resources (minions with their own motivations).

5. **V4: The Cracks**
   - Define: Blind Spot, The One Thing They Protect, The Tell, Redemption Potential (1-5).

6. **V5: The Web**
   - Connect the villain to the PCs: Why these PCs? Shared history? Temptation? Respect or contempt?

7. **V6: Escalation Arc**
   - Map: Introduction → Revelation → Confrontation → Retaliation → Climax.

8. **Generate Output**
   - Use the villain output template from `villain-mode.md`.
   - Validate with `validate_character_output.py` (villain mode auto-detected).
